# cleaningkit: What the Package Does (One Line, Title Case)

This package offers an automated toolkit for streamlining 4Mi survey
data cleaning and validation, enforcing protocols on data quality,
sampling, and enumerator performance. It provides a standardized
approach to checking interview consistency, identifying outliers,
"other" text responses, and detecting potential fraud based on official
SOPs.
